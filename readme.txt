####Le mot du créateur####
A l'occasion de la création d'un service d'Aide Online sur mon site www.Puissance-PC.net, j'ai en vain cherché un module capable de se connecter à IRC intégré a Xoops 2 entièrement paramétrable.
Après des recherches infructueuses, je me suis décidé à dévelloper ce module IRC, basé sur l'excellent applet PJIRC.
Dans cette version 1, assez basique il faut l'avouer, vous pourrez choisir le serveur ainsi que le port qui devra être connecté, ainsi que les canaux qui seront rejoints automatiquement.
Comme le nombre de langues actuellement disponible est très réduit, le choix s'effectue actuellement via l'admin.
Un petit bonus vous permettra de définir le message qui sera affiché quand la personne quittera le canal.
Le nickname est celui de Xoops. Les visiteurs se voient attribué automatiquement un pseudo Visiteur??? ou ??? est un nombre à 3 chiffres.

La version 2 comprendra la possibilité de redéfinir son pseudo sans avoir a utiliser la commande /nick, mais simplement grâce à un petit menu logé dans le bas droit de l'applet, choisir automatiquement de la langue, ajoute la traduction de la partie interface.


####Installation####
- Décompressez le dossier ircmod.
- Editez le fichier "background.gif" et mettez le logo de votre site, ou rendez le entièrement blanc.
- Transférez le dossier ircmod dans votre dossier modules de xoops sur votre serveur.
- Activez le module dans l'administration de xoops, et paramétrez à souhait!

####Installation instructions####
- Unpack ircmod repertory.
- Edit file "background.gif"
- Upload ircmod repertory on your server into the modules rep
- Activate module into your Xoops administration, and define your own settings. Don't forget to check group permissions!
- Have fun!

If anything is buggy, don't forget to contact me through email!
